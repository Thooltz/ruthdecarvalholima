# Site Institucional - Ruth de Carvalho Lima (Nova Versão)

Site institucional moderno e acolhedor desenvolvido para a advogada Ruth de Carvalho Lima, com foco em Direito Cível, Família, Sucessões e Previdenciário.

## 🎨 Características do Design

- **Estilo Acolhedor + Autoridade**: Design humano e profissional
- **Paleta de Cores**: Azul marinho (#1a365d) + Dourado (#D4AF37) + Branco
- **Tipografia Elegante**: Cormorant Garamond (títulos) + Inter (corpo)
- **Mobile-First**: Totalmente responsivo, otimizado para celular
- **Espaço em Branco**: Layout limpo e sofisticado
- **Botões Grandes**: Fáceis de clicar em dispositivos móveis

## 🚀 Como Usar

### Opção 1: Abrir Diretamente
Simplesmente abra o arquivo `index-novo.html` no seu navegador.

### Opção 2: Servidor Local (Recomendado)
Para melhor experiência:

```bash
# Com Python 3
python -m http.server 8000

# Com Node.js (http-server)
npx http-server

# Com PHP
php -S localhost:8000
```

Depois acesse: `http://localhost:8000/index-novo.html`

## 📱 Estrutura do Site

### 1. Hero (Topo)
- Título principal (H1): "Assessoria jurídica em Cível, Família, Sucessões e Previdenciário"
- Subtítulo acolhedor
- Badge: "Atendimento com sigilo e responsabilidade"
- Foto da advogada

### 2. Como posso te ajudar?
4 blocos detalhados com:
- **Direito Cível**: Contratos, cobranças, indenizações, questões com consumidores
- **Direito de Família**: Divórcio, guarda, pensão alimentícia
- **Sucessões**: Inventário, partilha, testamentos
- **Previdenciário**: Aposentadorias, revisão, BPC/LOAS

3. Acompanhamento e atualizações claras
4. Conclusão e orientação final

- Inventário: judicial ou cartório?
- Quanto tempo leva inventário?
- Quais documentos para aposentadoria?
- BPC/LOAS: quem tem direito?
- Revisão de benefício vale a pena?
- Como funciona a primeira consulta?

### 5. Sobre

- Foco em solução

## 🛠️ Personalização

### Alterar Cores
Edite as variáveis CSS no arquivo `styles-novo.css`:

```css
:root {
    --color-navy: #1a365d;
    --color-gold: #D4AF37;
    /* ... */
}
```

### Alterar Conteúdo
Edite diretamente o arquivo `index-novo.html` nas seções correspondentes.

### Alterar Imagem
Substitua o arquivo `public/images/ruth.jpg` ou altere o caminho no HTML.
