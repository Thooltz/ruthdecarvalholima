# Site Profissional - Ruth de Carvalho Lima

Site profissional desenvolvido para a advogada Ruth de Carvalho Lima, com foco em Direito Cível, Família, Sucessões e Previdenciário.

## 🎨 Características

- **Design Sóbrio e Elegante**: Cores preto, branco e dourado
- **Totalmente Responsivo**: Mobile-first, otimizado para todos os dispositivos
- **Performance Otimizada**: Carregamento rápido e animações suaves
- **SEO Otimizado**: Meta tags e palavras-chave estratégicas
- **Botão WhatsApp Fixo**: Acesso rápido ao contato em todas as páginas

## 📁 Estrutura de Arquivos

```
├── site.html          # Página principal HTML

```

## 🚀 Como Usar

### Opção 1: Abrir Diretamente
Simplesmente abra o arquivo `site.html` no seu navegador.

### Opção 2: Servidor Local (Recomendado)
Para melhor experiência e evitar problemas de CORS:

```bash
# Com Python 3
python -m http.server 8000

# Com Node.js (http-server)
npx http-server

# Com PHP
php -S localhost:8000
```

Depois acesse: `http://localhost:8000/site.html`

## 🛠️ Personalização

### Alterar Cores
Edite as variáveis CSS no arquivo `styles.css`:

```css
:root {
    --color-black: #000000;
    --color-white: #FFFFFF;
    --color-gold: #D4AF37;
    /* ... */
}
```

### Alterar Conteúdo
Edite diretamente o arquivo `site.html` nas seções correspondentes.

### Alterar Imagem
Substitua o arquivo `public/images/ruth.jpg` ou altere o caminho no HTML.
