# Site Institucional - Dra. Ruth Carvalho

Site institucional moderno, elegante e profissional para advogada, com estética minimalista e alto padrão (estilo "boutique law firm"), totalmente responsivo (mobile-first), com tema em lilás.

## 🎨 Características

- **Design Moderno**: Estética minimalista e profissional
- **Totalmente Responsivo**: Mobile-first, adaptável a todos os dispositivos
- **Tema Lilás**: Paleta de cores sofisticada
- **Tipografia Elegante**: Playfair Display para títulos, Inter para textos
- **Interatividade**: Menu mobile, FAQ accordion, animações suaves

## 📁 Estrutura de Arquivos

```
ruthdecarvalholima/
├── index.html      # Estrutura HTML principal
├── styles.css      # Estilos CSS com tema lilás
├── script.js       # JavaScript para interatividade
└── README.md       # Documentação
```

## 🚀 Como Usar

1. Abra o arquivo `index.html` em um navegador moderno
2. Ou hospede os arquivos em um servidor web

## 📱 Seções do Site

1. **Header**: Menu de navegação com logo e CTA WhatsApp
2. **Hero**: Apresentação principal com CTAs
3. **Áreas de Atuação**: Cards para Cível, Família e Sucessões, Previdência
4. **Sobre**: Informações profissionais e diferenciais
5. **Diferenciais**: Cards com pontos fortes
6. **Conteúdos**: Grade de artigos/posts
7. **FAQ**: Perguntas frequentes com accordion
8. **Contato**: Formulário e informações de contato
9. **Footer**: Informações finais e aviso legal
